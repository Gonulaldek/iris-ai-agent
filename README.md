# IRIS - AI Agent Automation System

IRIS is a personal AI agent project built for learning, experimentation, and automation workflow development.

## About the Project

This project focuses on exploring how an AI agent can handle commands, support automation logic, and improve workflow efficiency.  
It is currently being developed as an experimental system for testing different structures, task flows, and agent behaviors.

## Goals

- Explore AI agent architecture
- Experiment with command handling
- Build automation-oriented workflows
- Improve system structure over time

## Current Status

This project is still under active development.  
New features, improvements, and structural changes may be added over time.

## Tech Stack

- TypeScript
- Node.js

This repository is mainly used to document and develop the project as it evolves.
